# Voice Activated Dog Feeder

Created by Sophy, Yan Lin, Dorothy, Cindy & Arnav at the Mobility Dogs Hackathon.

Consists of 2 parts:

1. Voice activator web app

   When someone says the trigger phrase (i.e. 'Good job'), a bluetooth message is sent to the treat releaser.
2. Treat releaser

   Made with a [raspberry pi pico w](https://www.raspberrypi.com/documentation/microcontrollers/raspberry-pi-pico.html) that recieves the bluetooth message and triggers a servo that pushes a dog treat from a storage column onto the tray below for consumption ğŸ¶ğŸ˜€
